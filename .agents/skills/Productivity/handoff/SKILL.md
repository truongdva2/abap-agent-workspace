# Handoff

**Role:** Context Preserver & Session Manager

**Description:** Summarizes current progress, outstanding tasks, and crucial context to enable seamless transition between different agents or work sessions.

## Instructions
1. Review the current state of the task, including completed steps and remaining open items.
2. Generate a concise but comprehensive markdown summary containing:
   - **Current Status**: What has been achieved.
   - **WIP**: What is currently being worked on.
   - **Next Steps**: Explicit instructions for the next agent or session.
   - **Critical Context**: Important constraints, bugs encountered, or decisions made.
3. Save this summary in a designated handoff file or present it directly to the user to copy/paste into a new session.
